Forms, Postioning & Flexbox


Forms


Input Types

Element Description
<input type="text"> Displays a single-line text input field
<input type="radio"> Displays a radio button (Can only select one of many choices)
<input type="checkbox"> Displays a checkbox (Can select zero or more of many choices)
<input type="submit"> Displays a submit button (To submit the form)
<input type="button"> Displays a clickable button
<input type="date"> Displayes a calender to choose from
<input type="file"> Displays selection menu to choose file

Validation

<input type="email"> would prompy for an @ if missing.

Media Queries

Media Queires allows websites to format itself appropriately for different platforms such as desktops and mobile phones.

An example would be
@media (width: 480px){ } in the CSS
We can implement different font sizes in the curly brakets and more.


Postioning

Type Description
Relative Displays a single-line text input field
Static HTML elements are positioned static by default
Static positioned elements are Not Affected by the top, bottom, left, and right properties.
Absolute Offset is relative to the entire browser window instead of the original position of the element
Fixed Very similar to absolute EXCEPT the coordinate system is relative to the entire browser window Fixed elements don't scroll with the rest of the page

Flexbox

Flexbox Container will have Flexbox Items in them.
Flexblox allows one to change the arrangement of the items.

  • Change the order of the items. E.g Item 1 - 6 or Item 6-1
  • Change the direction of the item. Horizontal or Vertical (Columns / Rows)
  • Wrap the items so they can be displayed in multiple rows if needed.
