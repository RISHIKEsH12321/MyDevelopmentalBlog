JavaScript


Variables

It is a data container.

How to create and assign a value : var num = 12;

Rules to naming

  1. Has to begin with a letter or $
  2. Must not be a reserved word

Multiple Variable Creation:

  1. var item1 = "Item 1", item 2 = "Item 2";
  2. let item1 = 5;

Variable Types


String
  • Must be in single/double quotes (Must begin and end with the same type)
  • If multiple types of quotes are needed in string use \
  • \ can also be used to make another line
  • String can also be made using `string here`
  • Can be formatted using "string number ${a variable here}"
  • Can use methods or proerties to manipulate Strings : string.toUpperCase() or string.Length()

Numbers
  • Can have whole number, decimal or even negative number
  • Can be infinity if you use Infinity
  • NaN -> Not a Number
  • Can use methods or proerties to manipulate Strings : Math.random() or Math.round()

Booleans
  • Can be true or false (Case Sensitive)
  • Useful in Comparisons
  • Useful in Asking Questions in Code

Constant
  • Is an Immutable Variable
  • Can me made using const : const ID = S123087
  • Useful so that value of variable is not accidently changed

Objects

Is a kind of box with labeled folders.


Basics
  • Allows data to be Encapsulated
  • Create by var item = {label : "asd123dsa", "Color": "Pink"};
  • Containes keys and values

Manipulation
  • Get value using eky -> item.label or item["Color"]
  • Changing values -> item.label = 12 or item["Color"] = "Red"
  • Creating property -> item.texture = "Smooth" or item["texture"] = "Smooth"
  • Remove Property -> delete.texture or delete.item["texture"]

References
  • orginal2 = orginal -> Would mean that when original is changed, original2 would also be changed
  • To get a true copy -> copy = Object.assign([],original); or copy = {... orginal};

Arrays

To store lists of data that preserve their order


Basics
  • Create by var myArray = [1,2,'three','four',true];
  • Are zero indexed
  • Arrays can also contain Objects and Arrays -> [{name : "donkey"},[1,2,3]];
  • Can use properties and methods -> myArray.length

Manipulation
  • Getting Values -> myArray[0]
  • Replacing Values -> myArray[0] = 'One';
  • Adding Last Values -> myArray[myArray.length] = 'Last'
  • Adding Last Values -> myArray.push("Final")
  • Remove Last Values -> myArray.pop()
  • Remove/Addinng First Value -> myArray.shift()
  • Removing Middle Values -> myArray.splice(2,1)

Readbility
  • Whitespace -> Have Spaces to make long lines readable
  • Single Line Comments -> //Comment
  • Milti-line Comments -> /*Comment that is spread over multiple lines*/

Regular Expressions (Pattern)


Baics
  • Create by var pattern = /this/;
Uses
  • Returns true or false depending if pattern is in string1 -> pattern.test(string1);
  • Insensitive -> pattern.test(string1); where pattern = /this/i
  • Appears at the beginning -> pattern = /^this/i
  • Appears at the enf -> pattern = /this$/i

Comparisons & More


Comparisons
  • Strict Equality -> 1 === 1
  • Strict Inequality -> 1 !=== 1
  • Equality -> 1 == '1' gives true
  • Inequality -> 1 !== '1' gives false
  • Greater / Less than -> > and <
  • Greater / Less or Equal to -> >= or <=
Arithmetic
  • Addtion -> 1+2
  • Substraction -> 2-1
  • Multiplication -> 2*2
  • Division -> 2/2
  • Modulas -> 5%1 would give 1
  • Increment / Decrementing by n -> num += n / num -= n
  • Increment / Decrementing by 1 -> num ++ / num --
  • Contatanation -> 'cat' + 'dog' would give 'catdog'

Logical Operators
  • AND needs all conditions to be true -> true && false (returns false)
  • OR needed only one conditions to be true -> true || false (returns true)
  • AND is operated first, then OR
  • NOT changes true to false and vice versa -> !true (returns false)

Conditionals : If
  • How to use -> if (answer === true){ console.log('Yay')}
  • More Conditions -> else if (condition) {process}
  • More Conditions (Used if non of the conditions are used)-> else (condition) {process}

Conditionals : switch
  • Uses Cases
  • How to use -> switch(answer){
    case"true" : console.log("yay");
    break;
    case"false": console.log("boo");
    break;
    default: console.log("nothing");
    break;)

Terse ifs: One-liners
  • if (answer === true) {console.log('Yay')}
  • Truthy -> If string is calue = false, 0, -0, 0n, "", null, undefined, NaN
  • Falsy is opposite of Truthy

Terse ifs: Ternaries
  • How to use -> if true
    ? console.log('true')
    : console.log('false')

Type checking
  • Use by -> typeof variableName
  • More detailed -> typeof variableName === "string" (returns true)

Iterating with Loops


For Loops : Sequential
  • How to use -> for (let i = 0; i < 10; i++) {console.log(i)}
  • Loop through array ->
    for (let i = 0; i < arrayName.length; i++) {console.log(arrayName[i])}
  • Can also add if statements to cut the loop short and have a break in the statement

For Loops : Enumerative
  • Iterates over arrays
  • How to use (in) -> for (var P in pageNames) {console.log(P, pageNames[p])
  • How to use (of) -> for (var P in pageNames) {console.log(P, pageNames[p])
  • If looping through objects -> Use if (object.hasOwnProperty(count)){process) to make sure value is in the object and is not inherited.

Set
  • Needs constructor -> let mySet = new Set();
  • Sets stores value only once unlike array
  • Accessed and changed using special methods -> mySet.add('1') or mySet.has('1') === true

Map
  • Needs constructor -> let myMap = new Map();
  • Preserves the order of keys
  • Accessed and changed using special methods -> mySet.set('1','one') or mySet.has('1') === true

While Loop
  • Repeats until condition is met
  • How to use -> while (condition){process}
  • Improtant to have a break statement to prevent a infinite loop

Functions


Basics
  • How to use ->
    1.function name(parameters/argument) {process}
    2. const name(parameter/argument) = function { process }
  • Use retrun to give the processed value back to the main program
  • Default Value -> function showCase(value = "Default", count = 5){process}
  • Unknown number of parameter can be accessed -> function showCase(){arguments} where arguments is a array of parameters entered

Objects and Arrays in function
  • When passing a object as a parameter in a function, a copy of the memory address of the object is used in the function
  • This means that the object can be manipulated directly in the function as the function is using a referenceed version of the object unless the reutrned value is a copy

Functions as Parameters
  • Function can be made as an object and thus can be used as a paramenter in another function

Scopes
  • Global : Can be accessed and modified in all scripts
  • Can be used in functions or operators using global.varName
  • Local : Can be accessed and modified in only that script or function

Lamda Functions
  • Can be made using by --> const funcName = (parameter) => {process}

Rest (of the) Parameters
  • How to use -> function funcName(para1, para2, ...arrayName){process}
