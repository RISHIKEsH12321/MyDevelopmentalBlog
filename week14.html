Lottie Animations


What is Lottie?

A Lottie is a JSON-based animation file format that enables designers to ship animations on any platform as easily as shipping static assets. They are small files that work on any device and can scale up or down without pixelation


Reasons to use Lottie

  • Smaller Size
  • Increases Download Speed
  • Reduces Disk Space Utilised

Steps to Implement A Lottie Animation

  1. Find a lottie animation via this link
  2. Filter by Free
  3. Select Animation
  4. Save to Workspace to download
  5. Go to workspace (if not directed there)
  6. Select Handoff & Embed
  7. Modify Animation if you want via "Edit in Lottie Editor" Option in top bar
  8. Enable Asset Link
  9. Copy Embed HTML code
  10. Paste in Website

Done!

Working with Lottie Files using Code

  1. Place Lottie Player Javascript
  2. : Create a lottie player script in your HTML and reference the JSON file
  3. : Add in Javascript to determine how your Lottie JSON will work

Converting SVG to Lottie


LottieFiles provides a great SVG to Lottie tool

  1. Signup on LottieFiles
  2. Go to >> https://lottiefiles.com/svg-to-lottie
  3. Drag and drop any SVG Files (needs a LottieFiles account)
  4. Choose an animation type
  5. Save to your workspace to download

Getting SVG Files
