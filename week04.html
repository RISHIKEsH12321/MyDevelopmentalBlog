Responsive Images


Benfits of Responsive Images

  • Auto-resize based on screen size
  • Better Performance -> Load higher resolution images only when needed
  • Provides better design control depending on device

Using Responsive Images

  • Scale Up & Down : Using height/width = auto/ x%
  • If the image is Smaller than the div length : Use max-width = 100%
  • For different resoulutions : Use scrset = "{nultiple-urls}"
  • To fit a image in a box : Use background-size = cover
