HTML Elements

  • < html > : Created a HTML File
  • < head > : Defines the Head of the Site.
  • < body > : Definesthe Body of the Site.
  • < title > : Defines the Title of the Site.
  • < header > : Defines the Header of the Body.
  • < h1 >-< h6 > : Headers that increase and bold the size and thickness of the text.
  • < p > : Paragraphs always starts on a new line, and is usually a block of text.
  • < b > : Bold which bolds the text.
  • < em > : Italics the text.
  • < ul > : Creates an Unordered list.
  • < ol > : Creates an Ordered List
  • < li > : Are the items in a list.
  • < img src = "name.jpg"> : Creates a Picture.
  • < strong > : Gives text Importance.
  • < a href = "https://name.com" title = "This is a example "> : Provides a link to click on.
    • Youtube
  • < span > : Can be used to Style a Text.
    • Red text in Green Box
