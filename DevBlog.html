HTML Elements

  • < html > : Created a HTML File
  • < head > : Defines the Head of the Site.
  • < body > : Definesthe Body of the Site.
  • < title > : Defines the Title of the Site.
  • < header > : Defines the Header of the Body.
  • < h1 >-< h6 > : Headers that increase and bold the size and thickness of the text.
  • < p > : Paragraphs always starts on a new line, and is usually a block of text.
  • < b > : Bold which bolds the text.
  • < em > : Italics the text.
  • < ul > : Creates an Unordered list.
  • < ol > : Creates an Ordered List
  • < li > : Are the items in a list.
  • < img src = "name.jpg"> : Creates a Picture.
  • < strong > : Gives text Importance.
  • < a href = "https://name.com" title = "This is a example "> : Provides a link to click on.
    • Youtube
  • < span > : Can be used to Style a Text.
    • Red text in Green Box

Week 2

HTML Semantic elements

  • < header > : Defines the Header of the Body.
  • < nav > - Defines a set of Navigation links.
  • < section > - Defines a Section in a document.
  • < article > - Defines an independent, self-contained content.
  • < aside > - Defines content aside from the content (like a sidebar)
  • < footer > - Defines a footer for a document or a section.
  • < details > - Defines additional details that the user can open and close on demand

Cascading Style Sheets (CSS)

Simple CSS Statement
selector { property: value;} :
E.g p{color: purple;} changes the text color to purple.

How to implement CSS
  • Inline Style : < p style="text-align: left;" > (Inside the line of code)< /p>
  • Style Element : Write Sytle at the top of the HTML.
    Using style element and place CSS inside.
  • Link Element/External Style Sheet (Most Prefered Method)
    Links the CSS.cs to the program using < link rel="stylesheet" href="basic.css" type="text/css" >

ID and Class Attributes

  • ID : #IDname to include in CSS.
  • Class: .ClassName to include in CSS.

Typography

Font settings can be chaged in a myriad of ways.

  • < font-size > : Change Size of text.
  • < font-style > : Changes Style of font (Italic, Normal, Oblique)
  • < font-weight > : Changes Thickness of font.
  • < text-align > : Aligns text in relation to box. (Center, Left, Right, etc.)

Font Styles can also be taken from Google.
