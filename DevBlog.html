Postioning

Types of Postioning

  • Relative : Moves elements around relative to where they would appear on the page.
  • Static
  • Absolute : Offset is relative to the entire browser window instead of the original position of the element.
  • Fixed : Very similar to absolute EXCEPT the coordinate system is relative to the entire browser window. Fixed elements don't scroll with the rest of the page.

Z-Index
Z-index property lets you control the depth of elements on the page, like stacking elements on top of each other.


Flexbox

Flexbox Container will have Flexbox Items in them.
Flexblox allows one to change the arrangement of the items.

  • Change the order of the items. E.g Item 1 - 6 or Item 6-1
  • Change the direction of the item. Horizontal or Vertical (Columns / Rows)
  • Wrap the items so they can be displayed in multiple rows if needed.

Responsive Images

Benfits of Responsive Images

  • Auto-resize based on screen size
  • Better Performance -> Load higher resolution images only when needed
  • Provides better design control depending on device

Using Responsive Images

  • Scale Up & Down : Using height/width = auto/ x%
  • If the image is Smaller than the div length : Use max-width = 100%
  • For different resoulutions : Use scrset = "{nultiple-urls}"
  • To fit a image in a box : Use background-size = cover

How to create and assign a value : var num = 12;

  • var item1 = "Item 1", item 2 = "Item 2";

    Numbers

    Booleans

    Constant
