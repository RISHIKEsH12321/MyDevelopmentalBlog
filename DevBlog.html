HTML Semantic elements

  • < header > : Defines the Header of the Body.
  • < nav > - Defines a set of Navigation links.
  • < section > - Defines a Section in a document.
  • < article > - Defines an independent, self-contained content.

  • < footer > - Defines a footer for a document or a section.
  • < details > - Defines additional details that the user can open and close on demand

Cascading Style Sheets (CSS)

Simple CSS Statement
selector { property: value;} :
E.g p{color: purple;} changes the text color to purple.

How to implement CSS
  • Inline Style : < p style="text-align: left;" > (Inside the line of code)< /p>
  • Style Element : Write Sytle at the top of the HTML.
    Using style element and place CSS inside.
  • Link Element/External Style Sheet (Most Prefered Method)
    Links the CSS.cs to the program using < link rel="stylesheet" href="basic.css" type="text/css" >

ID and Class Attributes

  • ID : #IDname to include in CSS.

Typography

Font settings can be chaged in a myriad of ways.

  • < font-size > : Change Size of text.
  • < font-style > : Changes Style of font (Italic, Normal, Oblique)
  • < font-weight > : Changes Thickness of font.
  • < text-align > : Aligns text in relation to box. (Center, Left, Right, etc.)

Font Styles can also be taken from Google.

Week 3

Forms

There are many types of Inputs in Forms. Here are some of them.

  • < input type="text" > : Displays a single-line text input field
  • < input type="radio" > : Displays a radio button (Can only select one of many choices)
  • < input type="checkbox" > : Displays a checkbox (Can select zero or more of many choices)
  • < input type="submit" > : Displays a submit button (To submit the form)
  • < input type="button" > : Displays a clickable button
  • < input type="data" > : Displayes a calender to choose from.
  • < input type="file" > : Displays selection menu.

MEDIA QUERIES

Media Queires allows websites to format itself appropriately for different platforms such as desktops and mobile phones.

An example would be
@media (width: 480px){ } in the CSS. We can implement different font sizes in the curly brakets and more.

Postioning

Types of Postioning

  • Relative : Moves elements around relative to where they would appear on the page.
  • Static
  • Absolute : Offset is relative to the entire browser window instead of the original position of the element.
  • Fixed : Very similar to absolute EXCEPT the coordinate system is relative to the entire browser window. Fixed elements don't scroll with the rest of the page.

Z-Index
Z-index property lets you control the depth of elements on the page, like stacking elements on top of each other.


Flexbox

Flexbox Container will have Flexbox Items in them.
Flexblox allows one to change the arrangement of the items.

  • Change the order of the items. E.g Item 1 - 6 or Item 6-1
  • Change the direction of the item. Horizontal or Vertical (Columns / Rows)
  • Wrap the items so they can be displayed in multiple rows if needed.
